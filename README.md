# CYART-VAPT-REPORT
## 📌 Overview
This repository contains the complete Vulnerability Assessment and Penetration Testing (VAPT) engagement performed over a 4-week structured lab program aligned with:

- PTES (Penetration Testing Execution Standard)
- OWASP WSTG
- NIST SP 800-115
- OWASP API Top 10
- OWASP Mobile Top 10

---

## 🔍 Engagement Scope

| Week | Focus Area |
|------|------------|
| Week 1 | Security Assessment & Vulnerability Scanning |
| Week 2 | Active Scanning & Exploitation |
| Week 3 | Advanced Web Application Pentesting |
| Week 4 | API Security, Network Attacks & Mobile Pentesting |
